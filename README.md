# metal Desktop Edition

Browser-based AI orchestration cockpit. Pure C. localhost:5050.

## Build

```
make
```

## Run

```
./metal-desktop
```

Open `http://localhost:5050` in your browser.

## Part of metal

- **metal console** — CLI workflow system (Be/Beryllium)
- **metal desktop** — Browser-based cockpit (this repo)
